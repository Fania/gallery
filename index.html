This gallery can be customised for height, width, colours, images, and hyperlinks.

To do this, change the values of the variables in the :root selector in the CSS. Then, change the values in the list of images image_list in the JavaScript. Each value consists of two URL, the first is the path to the image, the second is the URL you'd like to link to.

This gallery works with swipe gestures on touch devices.
